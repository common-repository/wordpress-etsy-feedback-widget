== Description ==

This plugin adds a widget that displays the latest feedback from buyers for your Etsy shop.

== Installation ==

1. Upload the file wp-etsy-feedback.zip to your web server and extract the folder wp-etsy-feedback to your wp-content/plugins  directory.

2. Activate the plugin through the Plugins menu in Wordpress.

3. Set your Etsy shop name in the Etsy Feedback menu under the Settings menu in Wordpress.

4. Add the Etsy Feedback widget to your sidebar using the Widgets menu in Wordpress.

== Frequently Asked Questions ==

I have installed and activated the plugin but I don’t see it!

Check that you have added the widget to your theme

I get a message saying “set your id!”

You need to enter your Etsy shop name in the Etsy Feedback menu.

It’s not showing my latest feedback!

The plugin only checks for new feedback every 24 hours, to reduce the load on the Etsy server. Your new feedback will show up soon.
